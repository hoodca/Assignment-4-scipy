Your path string is pointing to the wrong folder name.

- In your code you use `.../downloads/INFO2/archive/...`
- Your actual file path is `.../Downloads/INFO 2/archive/...` (there is a space in `INFO 2`)
- Because of that mismatch, `pd.read_csv(file_path)` can’t find the file.

Use this:

```python
from pathlib import Path
import pandas as pd

file_path = Path(r"C:\Users\camjh\Downloads\INFO 2\archive\players_stats_by_season_full_details.csv")
print("Exists?", file_path.exists())  # should be True
df = pd.read_csv(file_path)
```
